This list shows online courses categorized by topics. I highly recommend to take them, and you can access the courses for free.

## Machine Learning
- [Machine Learning by Andrew Ng](https://www.coursera.org/learn/machine-learning)

## Internet of Things
- [Introduction to the Internet of Things and Embedded Systems by Ian Harris](https://www.coursera.org/learn/iot)
- [The Arduino Platform and C Programming by Ian Harris](https://www.coursera.org/learn/arduino-platform)
- [Interfacing with the Arduino by Ian Harris](https://www.coursera.org/learn/interface-with-arduino/)
- [The Raspberry Pi Platform and Python Programming for the Raspberry Pi by Ian Harris](https://www.coursera.org/learn/raspberry-pi-platform)
- [Interfacing with the Raspberry Pi by Ian Harris](https://www.coursera.org/learn/raspberry-pi-interface)

## Semantic Web
- [Web of Data by Catherine Faron Zucker et al.](https://www.coursera.org/learn/web-data)

## Text Mining
- [텍스트 마이닝 실전 및 분석](http://www.kmooc.kr/courses/course-v1:YSUk+FD_YSU_LIS01k+2021_T1/about)

## Writing Skill
